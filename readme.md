# Supermii_find_diff酷脸找碴

通过iOS平台上的[SuperMii 酷脸](https://itunes.apple.com/cn/app/supermii-ku-lian-zhuan-shu/id931181673)这个APP中的头像来做的找碴小游戏，主要是想尝试在微信中搞娱乐游戏，有兴趣的朋友可以用微信扫一下。

![supermii找碴](qrcode.png)

如果是iPhone4的话，估计会比较吃力，android系列的手机似乎应该还好，反正小屏幕的就是痛苦。

最后想说的是，APP作者当时给的头像找差异，真心难～

## 游戏构思

在**60秒**内找到图片中的不同，图片出现是随机的，并且不会重复出现。

## 朋友反馈

给几位朋友玩过之后，得到的结果都是：

* 游戏难度太高了……
* 时间太短了……
* 可玩性不高啊……

诸如此类的反馈，太多太多了，其实我也知道，如果真要做一个有可玩性的游戏，肯定是要设置关卡之类的，然后游戏难度值也要逐渐上升。不过这个仅仅只是个人为了娱乐而做的小东西，仅此而已……
